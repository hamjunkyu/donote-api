# Donote

> 개인 지출 관리 + 더치페이 정산을 하나의 플랫폼에서 제공하는 개인 재무 관리 백엔드 API

## Vision

"**Donote**"는 개인 지출 관리와 더치페이 정산 기능을 하나의 플랫폼에서 제공하는 **개인 재무 관리 백엔드 API**입니다. 기존 가계부 앱은 개인 지출 기록에만 집중하고, 기존 더치페이 도구는 1회성 금액 분배만 제공하는 것과 달리, Donote는 공동 지출 분배, 참여자별 정산 상태 추적, 미수금 관리, 정산 결과의 개인 통계 자동 반영까지 하나의 시스템에서 제공합니다. 이 시스템은 일상적으로 식사, 스터디, 구독 서비스 등의 비용을 함께 부담하면서도 "누가 누구에게 얼마를 받아야 하는지" 추적에 어려움을 겪는 **대학생 및 사회초년생**의 재무 관리 경험을 크게 개선할 것입니다.

## Goals

- 더치페이 정산 처리 시간을 수기 관리 대비 **80% 단축**
- CSV 업로드를 통해 100건 거래 등록 시간을 수기 입력 대비 **90% 절감**
- 월간 소비 리포트를 API 호출 즉시 제공하여 지출 현황 파악 시간 **90% 절감**

## Features

| 기능 | 설명 |
|---|---|
| 회원 인증 | JWT 기반 무상태 인증 |
| 지출/수입 관리 | 거래 CRUD, 필터링, 페이지네이션 |
| 카테고리 관리 | 시스템 기본 + 사용자 커스텀 카테고리 |
| 통계 | 기간별 합계, 카테고리별 비율, 월간 리포트 |
| 예산 관리 | 월별/카테고리별 예산 설정, 소진율 추적 |
| 더치페이 정산 | 균등/직접 분배, 회원·비회원 참여자 지원, 미수금 추적 |
| CSV Bulk Import | CSV 업로드, 해싱 기반 중복 방어, 카테고리 자동 매칭 |

## Tech Stack

- **Framework**: FastAPI (Python)
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy + Alembic
- **Auth**: JWT (python-jose + bcrypt)
